# WHMCS Bengali Language Translation

A community-driven, complete Bengali (bn_BD) language translation pack for WHMCS. This project aims to provide a native language experience for WHMCS users in Bangladesh and around the world.

## 🚀 Features
- Complete translation of WHMCS Client Area.
- Translation of Admin Area strings.
- SEO-friendly and community-maintained.
- Easy to install and customize.

## 📦 Installation Guide
1. Download the latest version of `bengali.php` from this repository.
2. Upload the file to your WHMCS installation directory: `/lang/`.
3. Log in to your **WHMCS Admin Area**.
4. Go to **Setup > General Settings > Localisation**.
5. Select **Bengali** from the Language dropdown.
6. Click **Save Changes**.

## 🛠 Usage of Overrides
To ensure your custom modifications are not lost during software updates, it is highly recommended to use the **Overrides** feature.
- Copy your custom translations to `/lang/overrides/bengali.php`.
- Visit the official [WHMCS Overrides Documentation](https://developers.whmcs.com/languages/overrides/) for more details.
