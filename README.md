# WHMCS Bengali Language Pack (bn_BD)

A complete, community-driven Bengali (bn_BD) language translation pack for WHMCS. This project aims to provide a native language experience for WHMCS users in Bangladesh and around the world, enabling seamless localization for both the Client and Admin areas.

## 🚀 Features
- **Full Localization**: Complete translation of the WHMCS Client Area.
- **Admin Support**: Includes translations for Admin Area strings.
- **Community Maintained**: Regularly updated to support the latest WHMCS versions.
- **Easy Integration**: Simple installation process for any WHMCS environment.
- **SEO-Optimized**: Clean and standard localization structure.

## 📦 Installation Guide
1. **Download**: Download the `bengali.php` file from this repository.
2. **Upload**: Upload the file to your WHMCS installation directory path: `/lang/`.
3. **Activate**:
    - Log in to your **WHMCS Admin Area**.
    - Navigate to **Setup > General Settings > Localisation**.
    - Select **Bengali** from the "Language" dropdown menu.
    - Click **Save Changes**.

## 🛠 Using Overrides (Recommended)
To ensure your custom modifications are safe and not overwritten during software updates, it is highly recommended to use the **Overrides** feature:
- Create a folder at `/lang/overrides/` if it does not exist.
- Copy your custom translations to `/lang/overrides/bengali.php`.
- Visit the official [WHMCS Overrides Documentation](https://developers.whmcs.com/languages/overrides/) for more details.
